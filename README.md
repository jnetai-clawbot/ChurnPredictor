# ChurnPredictor — Customer Churn Prediction & Analysis

**Job 57** | **Package:** com.jnetai.churnpredictor | **Repo:** https://github.com/jnetai-clawbot/ChurnPredictor

## What It Does
Track customer data and predict churn risk. Log interactions, calculate churn scores, and visualise customer health.

## Features
- Customer management with plan types
- Interaction logging (call, email, meeting, support)
- Churn risk scoring algorithm
- Customer retention dashboard
- Data export as JSON
- About page with version and update check

## Build
- Android API 26+, Kotlin, Room, KSP, ViewBinding + XML, Material3 Dark
- Uses Fragments + ViewModel + Navigation
- GitHub Actions CI with release APK
